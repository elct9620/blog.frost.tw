Container Summit 2014：未來的雲端

其實原本沒有打算參加的，不過最後還是禁不起誘惑選擇參加這場活動。

主要是因為有兩場議程是我希望在近期解決的問題，因此現場聽講肯定會是一個比較好的方法。

之後也有 Docker.Taipei 的活動，不過也沒有深入接觸，算是蠻可惜的⋯⋯

不過，這次活動結束之後我認為善用 Docker 應該會成為一個很有利的技術。
